# Documentos Referendados
# Grupo de Usuários Linux e Código Aberto  
## Estatística e Ciência de Dados – UFPR

Este repositório contém os **documentos institucionais oficiais** do Grupo de Usuários Linux e Código Aberto do curso de Estatística e Ciência de Dados da Universidade Federal do Paraná (UFPR).

Aqui estão mantidas as versões **ativas, revisadas e referendadas** dos documentos que regem a identidade, os princípios, a organização e o funcionamento do grupo.

---

## 📌 Propósito do Repositório

Este repositório tem como finalidade:

- Centralizar a documentação institucional do grupo  
- Garantir versionamento, rastreabilidade e transparência  
- Servir como fonte única de verdade para normas e diretrizes  
- Facilitar auditoria, revisão e evolução dos documentos  
- Assegurar alinhamento com práticas de ciência aberta e reprodutibilidade  

---

## 📜 Documentos Oficiais

Os principais documentos atualmente mantidos são:

- **Manifesto**  
  Define os princípios filosóficos e científicos do grupo  

- **Código de Conduta**  
  Estabelece normas de convivência e comportamento  

- **Carta de Fundação**  
  Formaliza a criação, objetivos e estrutura do grupo

---
Outros documentos poderão ser adicionados conforme a evolução do grupo.
---

## 🔄 Versionamento

Todos os documentos são versionados via Git. Alterações seguem princípios de controle rigoroso:

- Cada modificação deve ser registrada por commit  
- Mudanças relevantes devem ser discutidas previamente  
- Recomenda-se uso de pull requests para revisão coletiva  
- O histórico completo deve ser preservado  

Versões estáveis correspondem sempre ao estado da branch principal (`main`).

---

## 🤝 Processo de Contribuição

Embora este repositório contenha documentos oficiais, contribuições são bem-vindas, desde que sigam um processo estruturado:

1. Propor alterações via issue ou discussão  
2. Justificar tecnicamente a proposta  
3. Submeter pull request com as modificações  
4. Passar por revisão dos membros e/ou diretoria  
5. Aprovação e incorporação ao documento oficial  

Todas as contribuições devem estar alinhadas ao **Manifesto** e ao **Código de Conduta**.
